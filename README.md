# WWT COSMETICS - Premium Korean Skincare Hub

Official web platform for **WWT Cosmetics**, a global leader in authentic K-beauty distribution.

## Core Features
- **Wholesale Portal**: Bulk ordering for global retailers.
- **Premium Retail**: Curated individual skincare solutions.
- **Authorized Distribution**: Sourcing from 100+ prestigious Seoul brands (Anua, AXIS-Y, VT Cosmetics, etc.).

## Tech Stack
- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Interactions**: Embla Carousel + Lucide React
- **Animations**: Custom Luxe transitions

## Getting Started
1. `npm install`
2. `npm run dev`
